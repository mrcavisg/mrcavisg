   — zero manual work
  - **Enterprise SIEM/EDR** — Wazuh security monitoring on Oracle Cloud with Terraform, Prometheus, and Grafana — from

  - **B2B lead engine** — Scrapes Google Maps, analyzes reviews with Claude AI, generates personalized outreach —
  automated prospecting
  - **School management system** — Grade tracking, attendance, academic cycles — NestJS + Next.js serving teachers,
  students, and coordinators
